## Reference Priority

When looking for information:

1. **Local documentation** (this site, hpg-esm-tools)
2. **Official CTSM docs** (wiki, tech note, user guide)
3. **CESM forums** (community knowledge)
4. **Third-party sources** (verify carefully)
